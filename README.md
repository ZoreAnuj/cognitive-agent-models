# Cognitive Agent Models

A library of structured cognitive models and operations to enhance AI coding agents. It provides agents with mental frameworks for better reasoning, problem decomposition, and decision-making during development tasks.

## Key Features
*   **Mental Model Library:** Access to over 150 categorized mental models for strategic thinking.
*   **Orientation Detection:** Analyzes prompts to determine the required cognitive approach (e.g., analytical, creative, debugging).
*   **Cognitive Operations:** Implements structured reasoning steps like problem decomposition and solution validation.

## Tech Stack
Python, Pydantic, Custom prompt engineering for AI agents (Claude, GPT).

git clone https://github.com/zoreanuj/cognitive-agent-models.git
